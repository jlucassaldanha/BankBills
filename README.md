# BankBills API

## Descrição

A BankBills API é uma aplicação desenvolvida em C# .NET focada no processamento de extratos. O projeto foi estruturado utilizando conceitos de Clean Architecture, garantindo a separação clara de responsabilidades, facilidade de manutenção e escalabilidade para futuras integrações.

## Tecnologias Utilizadas

* **Linguagem/Framework:** C# / .NET
* **Roteamento:** Minimal APIs
* **Arquitetura:** Clean Architecture / Layered Architecture
* **Banco de Dados:** PostgreSQL
* **ORM / Acesso a Dados:** Entity Framework Core
* **Containerização:** Docker
